🌿 Pollution Complaint Management System

A database-driven web application that bridges the gap between citizens and environmental authorities by providing a structured digital channel for reporting pollution incidents.

🔍 About the Project

Every year, countless environmental violations go unreported due to the lack of an accessible, centralized system. This platform enables citizens to register pollution complaints with photo evidence, track complaint status in real time, and receive official responses from administrators — supporting UN SDGs 3, 11, 13, and 16.

✨ Features
🔎 Search complaints by pollution type, city, and PIN code

💬 Administrator reply system with complaint lifecycle tracking

📄 report generation for archival and legal purposes

🔐 Secure admin panel with login authentication
